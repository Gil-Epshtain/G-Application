 -+- Git Keep -+-
1. './source/' 		     - code and assets directory
2. './resources/'	     - assets for native: icons and splash
3. './config.xml' 	     - Cordova config
4. './package.json'      - npm config
5. './webpack.config.js' - WebPack config

 -+- Git Ignore -+-
1. './node_modules/      - Node auto generated according to 'package.json', run: $ npm install
2. './platform/          - Cordova auto generated according to 'config.xml', run: $ cordova platform add android|ios
3. './plugins/           - Cordova auto generated according to 'config.xml', run: $ cordova plugin add android|ios
4. './www/               - WebPack auto generated according to 'webpack.config.xml' and 'package.json', run: $ npm run dev|prod
