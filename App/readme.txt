Folderes structure

└─┬App // Project root
  |
  ├─┬node_modules
  | └ … // auto generated by npm (npn install) - gitignore
  |
  ├─┬platforms
  | └ … // auto generated by Cordova (cordovan platform add) - gitignore
  |
  ├─┬plugins
  | └ … // auto generated by Cordova (cordovan plugin add) - gitignore
  |
  ├─┬res // assets for Cordova (icon and splash)
  | |
  | ├─┬android
  | | |
  | | ├─┬icon 
  | | | └ … // Android’s app icon
  | | |
  | | └─┬splash
  | |   └ … // Android’s app splash
  | |
  | ├─┬iOS
  | | |
  | | ├─┬icon 
  | | | └ … // iOS’s app icon
  | | |
  | | └─┬splash
  | |   └ … // iOS’s app splash
  | |
  | └─┬orig
  |   └ … // original assets before resize  
  |
  ├─┬src
  | |
  | ├─┬App // Application code and assets
  | | |
  | | ├─┬Assets
  | | | └ … // App images, font, etc..
  | | |
  | | ├─┬Data
  | | | └ … // App data, strings, etc..
  | | |
  | | ├─┬Modules
  | | | └ … // Js modules, directives, services. etc..
  | | |
  | | ├Route.js // Angular’s config method
  | | |
  | | ├RunApp.js // Angular’s run method
  | | |
  | | └App.js // App entry point
  | |   
  | └index.html // View entry point
  |
  └─┬www
    └ … // auto generated by WebPack (output of WebPack, input of Cordova) - gitignore

Git Project Need only to contain:
1. ‘./src/‘ 		- code and assets directory
2. ‘./res/‘ 		- assets for native: icons and splash 
3. ‘./config.xml’ 	- Cordova config
4. ‘./package.json’ 	- npm config
5. ‘./webpack.config.js’ - WebPack config
